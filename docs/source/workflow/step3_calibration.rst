Step 3: Calibration (Optional)
==============================

If your shapes need to be aligned with the image, you can calibrate using one of two methods:

Load Calibration File
---------------------

- Click **"Load File"** in the Calibration panel
- Select a metadata/calibration file that contains coordinate mapping information

Manual Calibration
------------------

- Click **"Manual"** to enter manual calibration mode
- Right-click on the image to place 3 calibration points
- These points define the coordinate transformation between the XML and image coordinates
- Click **"Calibrate"** to apply the calibration
